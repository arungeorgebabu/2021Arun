BRANCHING:
git branch branchname : for creating new branch. eg: git branch alfa
git branch -a : for listing all branches in specified project.
git checkout branchname: to go inside into specific branches. eg: git checkout alfa
git push origin alfa: for pushing files from branch called alfa
git checkout master: for switching to master from any branch
git branch -d 'branch name': to delete a branch
git push origin --delete 'branchname'

DELETE:
To delete any file from local repository and cloud repository
git rm -r filename
git commit -m "msg"
git push origin master
